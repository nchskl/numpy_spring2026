# Лабораторная работа: Численные вычисления и анализ данных с использованием NumPy

## Описание

Лабораторная работа посвящена изучению основных возможностей библиотеки **NumPy** для решения задач численных вычислений, линейной алгебры и анализа данных. В проекте реализованы функции для работы с массивами, матрицами, статистического анализа и визуализации данных.

## Цели работы

- Освоить базовые операции с массивами и матрицами NumPy
- Изучить векторные и матричные операции
- Применить операции линейной алгебры (определители, обратные матрицы, решение систем)
- Провести статистический анализ данных
- Создать визуализацию результатов

## Реализованные функции

### 1. Создание и обработка массивов

| Функция | Описание |
|---------|---------|
| `create_vector()` | Создает вектор от 0 до 9 |
| `create_matrix()` | Создает матрицу 5×5 со случайными числами [0,1) |
| `reshape_vector(vec)` | Преобразует вектор (10,) в матрицу (2,5) |
| `transpose_matrix(mat)` | Транспонирует матрицу |

### 2. Векторные операции

| Функция | Описание |
|---------|---------|
| `vector_add(a, b)` | Сложение двух векторов |
| `scalar_multiply(vec, scalar)` | Умножение вектора на скаляр |
| `elementwise_multiply(a, b)` | Поэлементное умножение |
| `dot_product(a, b)` | Скалярное произведение векторов |

### 3. Матричные операции

| Функция | Описание |
|---------|---------|
| `matrix_multiply(a, b)` | Умножение матриц |
| `matrix_determinant(a)` | Вычисление определителя матрицы |
| `matrix_inverse(a)` | Вычисление обратной матрицы |
| `solve_linear_system(a, b)` | Решение системы линейных уравнений Ax = b |

### 4. Статистический анализ

| Функция | Описание |
|---------|---------|
| `load_dataset(path)` | Загрузка CSV файла в NumPy массив |
| `statistical_analysis(data)` | Расчет статистических показателей (средняя, медиана, стд. отклонение, мин, макс, перцентили) |
| `normalize_data(data)` | Min-Max нормализация данных |

### 5. Визуализация

| Функция | Описание |
|---------|---------|
| `plot_histogram(data)` | Построение гистограммы распределения оценок |
| `plot_heatmap(matrix)` | Построение тепловой карты корреляции предметов |
| `plot_line(x, y)` | Построение графика зависимости оценок от номера студента |
